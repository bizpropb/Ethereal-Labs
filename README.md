# Ethereal Labs - A Stunning React Website

Welcome to Ethereal Labs! This is a fully-featured, beautifully animated React website showcasing a fictional consciousness technology research institute.

## 🚀 Quick Start


### Run Development Server
```bash
npm install # Install dependencies
npm run dev # Run in DevMode
```
Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production
```bash
npm install # Install dependencies
npm run build # The production-ready files will be in the `dist` folder.
npm run preview # Run the production-ready preview
```

## ✨ What's Included

### Interactive Features
- **Particle System**: Interactive particles in the hero section that respond to your mouse
- **3D Graphics**: Rotating 3D sphere using React Three Fiber
- **Smooth Animations**: Every element animates beautifully using Framer Motion
- **Scroll Effects**: GSAP-powered scroll animations on the timeline
- **Data Visualizations**: Live charts showing research data
- **Hover Effects**: Delightful interactions on every card and button
- **Glass Morphism**: Modern frosted glass design throughout

### Sections
1. **Hero** - Eye-catching hero with particles and floating icons
2. **About** - Mission statement with interactive 3D sphere
3. **Research** - 6 research areas with detailed cards
4. **Technology** - Interactive tabs showcasing tech stack
5. **Data** - Beautiful charts and statistics
6. **Timeline** - Company milestones with scroll animations
7. **Team** - Team member profiles with social links
8. **Contact** - Contact form and information

## 📦 Deployment

### GitHub Pages

1. Update `vite.config.js` base path if needed
2. Build the project: `npm run build`
3. Deploy the `dist` folder to GitHub Pages

### Netlify/Vercel

Simply connect your repository and these platforms will auto-detect the Vite configuration!

## 🛠️ Customization

### Change Colors
Edit the CSS variables in `src/index.css`:
```css
:root {
  --primary: #00D9FF;
  --secondary: #B537F2;
  --accent: #FF2E97;
}
```

### Modify Content
Each section is a separate component in `src/components/`. Simply edit the content in these files!

### Add New Sections
1. Create a new component in `src/components/`
2. Import and add it to `src/App.jsx`
3. Add the corresponding CSS file
